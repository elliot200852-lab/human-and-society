# 人與社會教師社團網站 — Repo 規則

- 本 repo 是刻意公開的 GitHub Pages 網站，由 MyWork session 全權代管；可直接修改、commit、push `origin/main`，但各 repo 必須分開收尾。
- 網站維持 unlisted：所有頁面必須有 `noindex`，`robots.txt` 必須封鎖全站；除非 David 明示，不改成可搜尋。
- repo 只收文字、HTML/CSS/JS、設定與建置程式。圖片、PDF、影音、簡報、字型等二進位一律存 Google Drive，repo 只保存引用。
- public repo 禁止放入成員名單、學生／教師個資、內部行政資料、金鑰、service-account JSON、token 或 `.env`。
- Drive 只拉取已劃定為公開網站素材的資料夾；不得把工作原稿資料夾整批公開。
- push 失敗或發生衝突時停手回報；不強推、不改 remote、default branch 或 repo 可見性。
